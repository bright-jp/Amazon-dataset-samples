1000件を超える商品のAmazon datasetサンプルです。このdatasetは<b>Bright Data API</b>を使用して抽出されました。

<h2>この無料datasetに含まれるデータポイント:</h2>

* ```title```: 商品タイトル
* ```brand```: 商品ブランド
* ```description```: 商品の簡単な説明
* ```currency```: 商品の通貨
* ```availability```: 商品の在庫状況
* ```reviews_count```: レビュー数
* ```categories```: 商品カテゴリ
* ```asin```: 各商品の一意識別子
* ```buybox_seller```: buy box内の販売者
* ```root_bs_rank```: 一般カテゴリでのベストセラー順位
* ```images_count```: 画像数
* ```url```: 商品に直接リンクするURL
* ```video_count```: 動画数
* ```image_url```: 商品画像に直接リンクするURL
* ```rating```: 商品評価
* ```date_first_available```: 商品が最初に利用可能になった日付
* ```manufacturer```: 商品の製造元
* ```plus_content```: 追加コンテンツの有無を示す項目
* ```final_price```: 商品の最終価格
* ```buybox_prices```: 商品価格の詳細

さらに多数あります。

これは「Amazon Products (public data)」datasetから派生したサンプルsubsetであり、このdatasetには<b>269,400,000件以上の商品</b>が含まれています。

利用可能なdatasetファイル形式: <b>JSON, NDJSON, JSON Lines, CSV, または Parquet。必要に応じて、ファイルは.gzに圧縮できます</b>。

dataset配信タイプのオプション: <b>Email, API download, Webhook, Amazon S3, Google Cloud storage, Google Cloud PubSub, Microsoft Azure, Snowflake, SFTP</b>。

更新頻度: <b>1回, Daily, Weekly, Monthly, Quarterly, または Custom basis</b>。

抽出されたデータポイントに追加できるデータエンリッチメント: <b>リクエストに応じて対応。</b>

<b>[Amazon商品datasetの完全版を取得](https://brightdata.jp/products/datasets/amazon)</b>。

<h2>Amazon datasetsのユースケースは何ですか？</h2>

<h3>1. 在庫と価格戦略を最適化する</h3>

顧客が検索し購入している売れ筋商品を発見しましょう。在庫、価格設定、サプライチェーン、マーケティングのアプローチ全体を強化し、最大限の効率を実現します。

<h3>2. 競争優位性を獲得する</h3>

Amazonのトップセラーには成功の方程式があります。彼らの商品、レビュー、プロモーションを分析して、新しいビジネスアイデアの着想を得たり、自社のマーケットプレイスへの参加を促したりしましょう。

<h3>3. 消費者感情とブランドトレンドを理解する</h3>

さまざまな国でトレンドとなっているカテゴリやブランドをモニタリングし、消費者の嗜好や商品の人気の変化を検出します。
